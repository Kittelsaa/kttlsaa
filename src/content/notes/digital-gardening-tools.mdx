Digital gardening requires the right set of tools to flourish. This note explores the essential tools and techniques for maintaining a thriving [Digital Gardens](/notes/digital-gardens).

## Essential Tools

### 1. Note-Taking Systems

<RemoteImage
  src="https://res.cloudinary.com/dg3gyk0gu/image/upload/c_scale,w_1457/v1590193543/maggieappleton.com/notes/datatags.png"
  alt="Example of a note-taking system with colored tags"
  width="880px"
/>

The foundation of any digital garden is a robust note-taking system. Some popular options include:
- [Obsidian](/notes/obsidian) for local-first markdown
- [Roam Research](/notes/roam-research) for networked thought
- [Notion](/notes/notion) for all-in-one workspace

### 2. Publishing Platforms

Your garden needs a place to grow publicly. Some options:
- [Static Site Generators](/notes/static-site-generators) like Astro or Gatsby
- [Hosted Platforms](/notes/hosted-platforms) like Ghost or WordPress
- Custom solutions with [Bi-Directional Links](/notes/bi-directional-links)
